Installation

From NPM (Recommended)

The master branch is always published here. These builds are focus on stability and hence people should use this.

Provided you have a recent version of Node.js and NPM installed, run the command below.

npm install wwwserve --global

If you are running Linux or macOS and encounter a problem, try this one instead.

sudo npm install wwwserver --global
